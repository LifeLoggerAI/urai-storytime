# FINITE TIME — Canon and Shot Graph Contract

Parent authority: #27

This contract defines the private-by-default narrative authority for the FINITE TIME feature and the reusable URAI Film Foundry.

## Truth classes

`exact | approximate | family-memory | reconstructed | disputed | private | pending`

No generator, editor, narrator, or public surface may convert uncertainty into false precision.

## Required hierarchy

`project -> act -> sequence -> scene -> shot -> asset -> render -> review -> master`

Each node requires a stable ID, version, status, source-authority reference, privacy class, consent snapshot, sensitivity class, and approval lineage.

## Scene requirements

- narrative purpose
- chronology and truth class
- characters and age anchors
- location and time state
- dialogue / narration authority
- emotional arc
- dignity constraints
- public/private boundary
- required music, Foley, accessibility, and visual deliverables

## Shot requirements

- camera, lens, movement, duration, framing
- blocking, performance, wardrobe, props, environment, weather
- voiceover/dialogue timing
- music cue and sound events
- caption, transcript, audio-description, and haptic metadata
- render modality and dependencies
- continuity fingerprints
- rights, sensitivity, review, and promotion state

## Privacy boundary

Family photos, private addresses, medical/legal records, likeness source files, and sensitive memory evidence remain in private Drive authority. GitHub stores only opaque source IDs, classifications, hashes, and non-sensitive production metadata.

## First proof chapter

Farm -> family memories -> early digital worlds -> Lake O' the Pines -> Ski Nautique.

The proof chapter must pass full animatic, continuity, dignity, narration, score, sound, accessibility, provenance, and founder visual acceptance before feature-scale rendering is authorized.

## Completion gate

A scene is not production-ready until canon, consent, chronology, continuity, rights, budget, accessibility, and human review fields are complete. A render is not promotable until exact source identity, content hash, tool/model/prompt lineage, reviewer decision, and canonical storage path are retained.
